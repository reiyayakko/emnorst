## usage

詳しくは[ドキュメント](https://reiyayakko.github.io/emnorst)を参照してください。

### Browser

```html
<script src="https://unpkg.com/emnorst@0.0.1/dist/emnorst.umd.js"></script>
```

### NodeJS

install
`npm i emnorst`

```javascript
// CommonJS
const emnorst = requie("emnorst");

// ES2015 Module
import * as emnorst from "emnorst";

emnorst.isNumber(1);
// > true
```
