## usage

詳しくは[ドキュメント](https://reiyayakko.github.io/emnorst)を参照してください。

パッケージの種類は以下の通りです。

- commonJS (cjs)
- ES2015 Module (es)
- umd (umd)

### example

ブラウザで使用する場合は以下のようなタグを埋め込んで使用してください。

```html
<script src="https://cdn.jsdelivr.net/gh/reiyayakko/emnorst@1.1.1/dist/reiyayakko.umd.min.js"></script>
```
